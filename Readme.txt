This project is a simulator to make an order online in a market

It is written using java netbeans and it is implemented by using the servlet


to open the database you must open a terminal and go in the directory of java and vrite this command

java -cp hsqldb.jar org.hsqldb.Server -database.0 file:spesa -dbname.0 xdb

for example

C:\Sun\SDK\jdk\bin\java -cp hsqldb.jar org.hsqldb.Server -database.0 file:spesa -dbname.0 xdb
